# 🔐 Flask Login System

A secure login system built using Flask, SQLite, and Werkzeug password hashing.

---

## 🚀 Features

- User Signup & Login
- Password Hashing (secure)
- Session Management
- Profile Page
- Edit Profile
- Delete Account (with "type DELETE" confirmation)
- Flash Messages
- Responsive UI with Glassmorphism design

---

## 🛠 Tech Stack

- Python
- Flask
- SQLite
- HTML/CSS
- Werkzeug Security

---
