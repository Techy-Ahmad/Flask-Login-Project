# 🔐 Flask Authentication System

A secure and fully functional Flask-based authentication system with user registration, login, session management, profile handling, and account deletion. The project uses SQLite as the database and Werkzeug for password hashing.

---

## 🚀 Live Demo
https://techyahmad.pythonanywhere.com

---

## ✨ Features

- User Signup & Login system
- Secure password hashing (Werkzeug)
- Session-based authentication
- Profile page with user details
- Edit profile functionality
- Account deletion with confirmation
- Flash messages for user feedback
- SQLite database integration
- Clean and responsive UI with animations

- SQLite

- Werkzeug Security
