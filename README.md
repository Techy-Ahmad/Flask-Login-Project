Flask Login System 🔐

This is a beginner-friendly web application built using Flask that demonstrates the fundamentals of full-stack web development. The project implements a simple login system where users can enter a username and password, which are then validated on the backend using Python. Based on the input, the user is redirected to either a success page or a failure page. The project also includes a basic frontend built with HTML and CSS to provide a clean and user-friendly interface.

This project helped in understanding key web development concepts such as routing, handling form data, template rendering, and connecting frontend with backend logic using Flask.

🚀 Features
Simple login page interface
Username and password authentication
Success and failure response pages
Clean and styled UI using CSS
Flask routing with redirects
Basic full-stack project structure

🛠️ Tech Stack
1.Python
2.Flask
3.HTML
4.CSS
